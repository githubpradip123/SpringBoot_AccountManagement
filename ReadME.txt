# Spring Boot Backend Project:-

     1. Prerequisites:
        1) Java (version [Specify required version])
        2) Maven or Gradle
        3) A database (e.g., PostgreSQL, MySQL) with appropriate credentials configured in application.properties.

     2. Installation:
         1) Clone the repository: git clone https://github.com/your-username/SpringBoot_AccountManagement.git
         2) Navigate to the project directory: cd SpringBoot_AccountManagement
         3) Install dependencies: Maven: mvn clean install

     3. Running the App:
          1) Start the application: Maven: mvn spring-boot:run
          2) The API will be accessible at the specified port (e.g., http://localhost:8080).

     4. Accessing H2 Console:
          1) The H2 database console is available at http://localhost:8080/h2-console.
          2) Enter the database credentials configured in application.properties.
             Example: Enter the following credentials:
                      JDBC URL: jdbc:h2:mem:testdb
                      Username: sa
                      Password: (leave blank)
                      Click the "Connect" button to access the H2 console.

     5. Swagger UI:
          1) The Swagger UI is available at http://localhost:8080/swagger-ui/index.html
